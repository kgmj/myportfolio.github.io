PROJECT 1

Project brief: In this project I used Python to do a basic data analysis of a spreadsheet using fake sales data from a csv file.

  • Spreadsheet Analysis

What were my required tasks:
1) To read all the data from the spreadsheet
2) Collect all the sales from each month into single list
3) Output the summary of the sales to a graph using matplotlib and pandas
4) Output the month with the highest sale
5) Output the month with the lowest sale

Issues/challenges I faced:
a) Making a graph for the first time
b) Outputting the percentage changes of the sales data

How did I overcome these challenges:
a) As this was the first Python project I had ever created, having to output the data to a graph was initially quite daunting. For this task, I used online resources to help me create the first draft of the graph and then when I felt more comfortable, I began to experiment with both the colour and style of graph.

b) The issue was that the percentage changes would be output to the console in a table but, they wouldn’t be to one decimal place instead it would display all the figures after the decimal point. Even when I tried to use the round() function it still wouldn’t work, so I ended up having to remove this task from the project as it just wouldn’t work.

Results: All the tasks were completed and can now be found on my GitHub and is also linked below!
